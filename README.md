# Pubmarks Datasets

A repository of daily updated datasets with data from the public markets.

## Data

### Stocks

Daily OHLCV data in CSV, combined and per calendar year.

| Ticker | OHLCV Period |
| ------ | ------------ |
| AAPL | [1980–2026](data/stocks/AAPL/ohlcv.csv) |
| ADBE | [1986–2026](data/stocks/ADBE/ohlcv.csv) |
| CRM | [2004–2026](data/stocks/CRM/ohlcv.csv) |
| INTU | [1993–2026](data/stocks/INTU/ohlcv.csv) |
| MSFT | [1986–2026](data/stocks/MSFT/ohlcv.csv) |
| RMD | [1995–2026](data/stocks/RMD/ohlcv.csv) |

## CDN

All files are available via GitHub Pages at **https://pubmarks.github.io/datasets/**. 

Use these URLs directly in your scripts or applications - GitHub Pages is a reliable CDN with global edge caching, no auth required, and no rate limits for public data.

Example:

```
https://pubmarks.github.io/datasets/stocks/AAPL/ohlcv.csv
```
