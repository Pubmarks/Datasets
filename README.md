A repository of daily updated datasets with data from the public markets.

`combined.csv` merges OHLCV and EPS into a single daily series (`date, open, high, low, close, volume, ttm_net_eps, pe_ratio`); EPS columns are empty for dates before EPS coverage begins.

`avgpe_5.json` contains P/E statistics (mean, median, mode, Shiller CAPE, harmonic mean, and more) computed over the trailing 5 years.

## CDN

All files are available via GitHub Pages. 

Use these URLs directly in your scripts or applications - GitHub Pages is a reliable CDN with global edge caching, no auth required, and no rate limits for public data.

Examples:

```
https://pubmarks.github.io/datasets/stocks/AAPL/ohlcv.csv
https://pubmarks.github.io/datasets/stocks/TSLA/eps.csv
https://pubmarks.github.io/datasets/stocks/NVDA/combined.csv
https://pubmarks.github.io/datasets/stocks/MSFT/avgpe_5.json
https://pubmarks.github.io/datasets/stocks/TSM/avgpe_10.json
```
